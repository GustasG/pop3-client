# pop3-client

POP3 client that was created for computer networks course.

This client was implemented according to [RFC1939](https://www.ietf.org/rfc/rfc1939.txt) specification


## Implemented commands

- USER
- PASS
- QUIT
- NOOP
- STAT
- LIST [msg]
- DELE
- RSET
- UIDL [msg]
- RETR

## Missing commands

- TOP
- APOP
